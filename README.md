# R tools from [@hinstitute](http://www.twitter.com/hinstitute) #
###_A suite of tools for R maintained by [Harmony Institute](http://www.harmony-institute.org/)'s data lab_###
How to Install:

	library("devtools")
	install_github("hiR", "hinstitute")
	library("hiR")
	help(package="hiR")
